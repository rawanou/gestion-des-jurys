# Module Odoo – Gestion des Jurys

 Description du projet

Le module **Gestion des Jurys** est un module personnalisé développé sur **Odoo 17**.  
Il permet de gérer les **jurys académiques** ainsi que les **décisions prises par ces jurys** (Admis, Ajourné, Refusé), dans un environnement ERP structuré.

Le projet est déployé à l’aide de **Docker** afin de garantir une installation simple, reproductible et indépendante du système.

---
 Objectifs du module

- Gérer les **jurys** (sessions, années universitaires)
- Enregistrer les **décisions du jury**
- Assurer la **traçabilité des délibérations**
- Centraliser les informations académiques
- Appliquer les bonnes pratiques de développement Odoo

---

 Technologies et outils utilisés

- **Odoo 17**
- **Python 3**
- **PostgreSQL**
- **Docker & Docker Compose**
- **XML (vues Odoo)**
- **VS Code** (éditeur de code)
